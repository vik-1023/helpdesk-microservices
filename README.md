\# Helpdesk Ticketing System - Microservices Architecture



\## 📌 Modules



\- API Gateway (JWT + RBAC Security)

\- Auth Service (Login / Signup)

\- User Management Service

\- Service Registry (Eureka)



\## 🛠 Tech Stack



\- Spring Boot

\- Spring Cloud Gateway

\- Spring Security

\- JWT Authentication

\- MySQL

\- Microservices Architecture



\## 🔐 Security Features



\- JWT Authentication

\- Role-Based Access Control (ADMIN / AGENT / USER)

\- API Gateway Level Authorization

\- Service-level Ownership Validation (in progress)



\## 🚀 Architecture



Microservices-based backend system using:



\- API Gateway routing

\- Eureka Service Discovery

\- Centralized Authentication

\- Role-based ticket access control
